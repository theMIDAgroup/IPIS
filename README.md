# IPIS – Repository

Repository with all the software of the IPIS project

### Description
AGquad: A MATLAB package for 1D and 2D anti-Gauss rules (https://github.com/CaNAgroup/AGquad)

dextre: Decorrelation stretch for a multispectral image (https://github.com/CaNAgroup/dextre)

Glyph Creator: A Tool for the Construction of Synthetic Petroglyphs. (https://github.com/CaNAgroup/GlyphCreator)

mngnreg: Minimal norm (regularized) solution of a nonlinear least-squares problem (https://github.com/CaNAgroup/mngnreg)

psdatasynth: A Matlab package for generating synthetic datasets for photometric stereo (https://github.com/CaNAgroup/psdatasynth)

splitImageFract: A function for exctracting a glyph embedded in an underlying background surface using ADMM and Fractional Laplacian (https://github.com/CaNAgroup/splitImageFract)

Matlab codes for the NPD, NPDIT and PNPD methods developed within IPIS (https://github.com/SAleotti/-P-NPD-IT-)

Python codes for the PNPD method (https://github.com/Giuseppe499/PNPD)

Codes for GraphLa+Ψ (https://github.com/devangelista2/GraphLaPlus)

Codes for iterated GraphLa (https://github.com/pimauri/it_graphLa)

Codes for iterated Arnoldi-Tikhonov method (https://github.com/davide-furchi/iterated-Arnoldi-Tikhonov)

Sparsity-Enhancing Desaturation algorithm (SE-DESAT) (https://github.com/SabrinaGuastavino/Adaptive_SE_DESAT\0)
